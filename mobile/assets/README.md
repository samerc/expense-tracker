# App Assets

Place the following image files in this folder:

## Required Files

- `icon.png` - App icon (1024x1024 px)
- `splash.png` - Splash screen image (1284x2778 px recommended)
- `adaptive-icon.png` - Android adaptive icon foreground (1024x1024 px)

## Generating Assets

You can use tools like:
- [Expo Icon Builder](https://buildicon.netlify.app/)
- [App Icon Generator](https://appicon.co/)
- Figma or any image editor

## Color Scheme

Primary color: #3B82F6 (blue)
This is used as:
- Splash background
- Adaptive icon background
- Tab bar active color
